# 💬 Guia de Vendas & Estratégia de Conversão

Este guia descreve como usar a landing page como ferramenta de vendas e como converter clientes potenciais em clientes reais.

## 🎯 Estrutura de Vendas

### 1. A Jornada do Cliente

```
Atração (Landing Page)
    ↓
Educação (Seções da Página)
    ↓
Consideração (Por Que Escolher)
    ↓
Decisão (CTA → WhatsApp)
    ↓
Fechamento (Conversa no WhatsApp)
```

### 2. Funnel de Conversão

| Etapa | Ação | Taxa Esperada |
|-------|------|--------------|
| Visitante | Acessa página | 100% |
| Engajado | Scrolls por + 30s | 40-60% |
| Interessado | Clica em algum CTA | 5-15% |
| Lead | Entra em conversa WhatsApp | 100% (dos que clicam) |
| Cliente | Fecha proposta | 30-50% |

**Se o cliente não especificar:**

```

   a) Sim

```

```

```

#### 2. Cores do Botão

- Laranja (atual) = Ação/Atenção ✓
- Verde = Sucesso (teste)
- Vermelho = Urgência (teste)

#### 3. Posicionamento

- Topo direito (atual) ✓
- Meio da página (teste)
- Flutuante fixo (teste)

```
Taxa de Clique (CTR):
- Esperado: 5-15% dos visitantes
```

## 📈 Projeção de Crescimento

### Mês 1 (Lançamento)

```
🎯 Meta: 5 instalações

Ações:
  • Compartilha landing page no WhatsApp (contatos)
  • Publica em grupos de condomínios
  • Pede indicações de amigos
  • Faz pós em redes sociais
  
Resultado esperado: 2-5 clientes
```

### Mês 2-3 (Tração)

```
🎯 Meta: 10-15 instalações/mês

Ações:
  • Clientes antigos indicam amigos
  • Reviews & depoimentos na landing
  • Anúncios no Facebook/Instagram
  • Campanhas de email marketing
  
Resultado esperado: 8-12 clientes/mês
```

### Mês 4-6 (Escala)

```
🎯 Meta: 20-30 instalações/mês

Ações:
  • Programa de indicação ativo
  • Parceria com corretoras de imóveis
  • Anúncios regionais
  • Video marketing no YouTube
  
Resultado esperado: 15-25 clientes/mês
```
