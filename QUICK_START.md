# ⚡ Quick Start - Comece Agora em 5 Minutos

Guia ultra-rápido para colocar a landing page no ar em minutos.

---

## 🚀 Passo 1: Configurar WhatsApp (2 min)

1. Abra o arquivo `script.js` com bloco de notas
2. Procure por esta linha:
   ```javascript
   const WHATSAPP_NUMBER = '5511999999999';
   ```
3. Substitua `5511999999999` pelo **seu número**
4. Formato: `55` + DDD + número (ex: `5521987654321`)
5. Salve o arquivo (Ctrl+S)

✅ **Pronto!**

---

## 🧪 Passo 2: Testar Localmente (1 min)

**Windows:**
```cmd
cd C:\Caminho\Para\Land-page
python -m http.server 8000
```

**Mac/Linux:**
```bash
cd ~/Documentos/Land-page
python -m http.server 8000
```

Abra no navegador: `http://localhost:8000`

✅ **Pronto! A página está rodando localmente!**

---

## 📤 Passo 3: Colocar Online (2 min)

### Opção A: Netlify (Mais Fácil)

1. Acesse [netlify.com](https://netlify.com)
2. Clique "Drag and drop your site folder here"
3. Arraste a pasta `Land-page`
4. **Pronto! Sua página está online!**

URL gerada: `https://seu-nome-aleatorio.netlify.app`

### Opção B: GitHub Pages

1. Crie conta em [github.com](https://github.com)
2. Crie novo repositório `safe-gate-landing`
3. Clique "Upload files"
4. Arraste os arquivos
5. Vá em Settings > Pages > Selecione `main` branch
6. **Pronto! Sua página está em `https://seu-usuario.github.io/safe-gate-landing`**
