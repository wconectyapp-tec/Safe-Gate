## 🚀 Próximos Passos (Imediatos)

### **PASSO 1: Configure (2 min)**

Abra `script.js` e procure por:
```javascript
const WHATSAPP_NUMBER = '5511999999999';
```

Substitua pelo seu número:
- Formato: `55` + DDD + número
- Exemplo: `5521987654321`

### **PASSO 2: Teste (1 min)**

```bash
# Na pasta Land-page
python -m http.server 8000
# Acesse: http://localhost:8000
```

### **PASSO 3: Publique (2 min)**

Escolha uma opção:
- **Netlify:** Drag & drop (mais fácil)
- **GitHub:** Envie + ative Pages
- **Vercel:** Conecte GitHub
- **Firebase:** CLI de upload
